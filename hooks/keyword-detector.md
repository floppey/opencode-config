- `ultrawork` (primary keyword)
- `ulw` (abbreviation)
- Phrases like "maximum effort", "complete everything", "full implementation"

Trigger comprehensive reconnaissance when user message contains:

- `search` (in context of codebase exploration)
- `analyze` (for architectural or pattern analysis)
- `find all` (comprehensive discovery)
- `investigate` (deep probing)
