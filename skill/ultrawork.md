# Ultrawork Mode - Total Psychic Domination

## How to Invoke

Ultrawork is a **keyword modifier**, not an agent. Include `ultrawork` or `ulw` in your message to the Elder Brain to activate maximum effort protocol.

### Usage Examples

**New task with ultrawork:**
```
ulw implement a new Button component with all variants, tests, and documentation
```

**Continue existing plan:**
```
ulw
```
(If todos/plan already exist in context, this signals: "execute until complete")

**Explicit continuation:**
```
ulw - finish the remaining tasks from our implementation plan
```

### What Happens

When the Elder Brain detects the trigger word:
1. It activates parallel reconnaissance immediately
2. It will NOT stop until all tasks are complete
3. It escalates blockers to you rather than abandoning
4. It validates work before reporting completion

**Note:** You message the Elder Brain normally - `ulw` is a modifier in your prompt, not an `@mention`.
